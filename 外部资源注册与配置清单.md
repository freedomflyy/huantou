# 幻头 AI 小程序 - 外部资源注册与配置清单

最后更新：`2026-03-12`

---

## 1. 现在就需要准备（P0）

### 1.1 微信小程序账号（必须）
- 用途：真实登录、域名配置、发布审核
- 你需要完成：
  - 创建/确认小程序主体
  - 进入后台拿到 `AppID`
  - 生成并保存 `AppSecret`
- 后端对应配置：
  - `AUTH_LOGIN_MODE=wechat`
  - `WECHAT_APPID=...`
  - `WECHAT_APP_SECRET=...`

### 1.2 可访问的 HTTPS 域名（必须）
- 用途：小程序调用后端 API
- 要求：
  - `https`
  - 稳定公网域名
  - 国内上线建议完成备案（涉及微信合法域名与发布稳定性）
- 微信后台需要配置：
  - request 合法域名（API）
  - upload/download 合法域名（后续图片上传下载）
  - 当前生成图实际走 COS 时，还要加入：
  - `https://huanto-1331012038.cos.ap-beijing.myqcloud.com`

### 1.3 测试环境公网映射（开发期可选）
- 用途：本地开发联调
- 建议：先用映射工具联调，正式上线切云服务器
- 说明：映射可用于开发，不建议作为生产长期方案

---

## 2. 第二阶段准备（P1）

### 2.1 对象存储（建议腾讯云 COS）
- 用途：保存生成图片、缩略图、7天过期清理
- 你需要提供：
  - `COS_SECRET_ID`
  - `COS_SECRET_KEY`
  - `COS_BUCKET`
  - `COS_REGION`
- 后端对应配置：
  - `STORAGE_PROVIDER=cos`

### 2.2 火山引擎图像能力（可选但建议）
- 用途：替换本地 SD/ComfyUI，降低算力维护成本
- 你需要提供：
  - `VOLCENGINE_API_KEY`
  - `VOLCENGINE_ENDPOINT`
- 后端对应配置：
  - `AI_PROVIDER_DEFAULT=volcengine`

---

## 3. 上线前必须准备（P2）

### 3.1 内容安全审核服务
- 用途：文本/图片审核，满足发布和风控要求
- 配置方式：
  - 先保留 `MODERATION_PROVIDER=mock`
  - 上线前切真实服务（腾讯云或火山引擎）

### 3.2 运维基础
- 数据库备份策略
- 日志告警（建议 Sentry + 服务监控）
- 图片过期清理任务（7天）
