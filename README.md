# 幻头 AI 头像生成小程序

这是一个毕业设计项目，目标是做一套可联调、可演示、可继续扩展的 AI 头像生成产品原型。仓库包含：

- `backend`：基于 `FastAPI` 的后端服务，负责登录、积分、任务、资产、审核与管理接口
- `miniprogram_v2`：微信原生小程序前端，负责创作流程、结果展示、资产管理与个人中心

## 项目定位

这个项目不是单页 Demo，而是一个完整的前后端联动原型，覆盖了从登录到生成、从结果到资产沉淀的闭环。

核心能力包括：

- 微信登录与会话管理
- 文生图、图生图、风格迁移、快速编辑
- 积分消耗与积分流水
- 作品列表、收藏、历史记录
- 内容审核与基础管理接口

## 技术栈

## 业务主链路

1. 用户进入小程序并完成登录
2. 创建生成任务并扣除积分
3. 后端异步消费任务
4. 小程序轮询任务状态
5. 生成结果沉淀到资产列表
6. 用户可继续收藏、查看历史或再次编辑
